CSE 270: Software Quality Assurance Testing

Welcome to CSC 270
Software Quality Assurance Testing

Overview

Explore Testing in relation to S/W Development, Business, Career, System Architecture, Quality, and Security. Appraising different tools (manual to automation), process, and techniques. Create Test Plans & Processes scaling from unit to system.

Release Notes

Objectives

By the end of the week, each student will be able to:

  • Argue how Testing relates to S/W Development, Science and Engineering activities
  • Classify Professional, Business, Career field, Security Issues
  • Appraise different testing tools and justify which to use in a given scenario
  • Assess the different Testing Techniques and discriminate the different use of each
  • Prepare and Implement Test Plans at different Levels using different Processes and Tools
