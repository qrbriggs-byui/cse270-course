CSE 270: Classroom Notes M04: Software Techniques

Notes for Classroom

Overview

There are several types of testing techniques, including exploratory, domain, flow, and box testing. There are techniques that run the throughout the lifecycle (later week), provide different approaches, and coverage, based on the need and purpose of the software.

  • This week students will explore and present on different techniques and the pros and cons of each.

Objectives

By the end of the week, each student will be able to:

  • Summarize Testing Techniques (Exploratory, Domain, Flow, Box, etc.)

    • 2 Post will contain how 1 of the testing techniques can be used to verify the feature

      • A testing technique that can be applied to a different feature or
      • A different technique that can be applied to the same feature.
