CSE 270: W04 Quiz Questions: Testing Techniques

The final deliverable is to come up with three quiz questions drawn from your presentation. With these questions, try to capture the "core lessons" to be learned from your presentation. In other words, don't bother to create a question that is trivial. Focus instead on the information worth learning. What are the three key takeaways from your topic?

These three quiz questions will be submitted separately on the next assignment in this lesson. Please make sure to clearly indicate the correct answer. Each question must be one of these types:

  • True/False (one correct answer or one incorrect answer)
  • Multiple choice (one correct answer, many incorrect answers)
  • Multiple select (n-correct answers, many incorrect answers)
  • Matching (1-to-1 or n-to-1 or n-to-m) [auto grade]

It is easy to formulate fact-based questions. However, you should create comprehension or reasoning questions:

Fact-Based Questions

Fact-based questions are easy questions. Is it equal or not equal to the definition or situation. Often it is possible to correctly answer a fact-based question without understanding the material. The following are examples of fact-based questions:

  • True/False: Statement of fact. For example:
    The * symbol is used for multiplication.
  • Multiple Choice: Selection of the correct answer. For example:
    Which of the following symbols corresponds to multiplication?
    A. -
    B. +
    C. *
    D. /
  • Matching: Match a name with a description, much like several multiple choice questions combined in one. For example:
    Match the name with the definition:
    A. -
    B. +
    C. *
    D. /    		 1. Division
    2. Addition
    3. Subtraction
    4. Multiplication

Comprehension questions require the subject to understand the problem fully. They cannot be answered by searching for facts. The facts must be understood first. The following are examples of comprehension-based questions:

  • True/False: Using comparison or understanding of facts. For example:
    Multiplication is like performing several additions.
  • Multiple Choice: Selection of the correct answer. For example:
    Which of the following symbols would be useful in find the number of apples in a bag?
    A. -
    B. +
    C. *
    D. /
  • Matching: Once again, like several multiple choices combined. For example:
    Match the scenario with the mathematical operator:

    A. -
    B. +
    C. *
    D. /

    1. Subdivide into groups
    2. Find the number of apples in a bag
    3. Determine how many are left after some are removed
    4. Tabulate by groups

Reasoning Questions

Reasoning questions require the subject to understand rules and then apply them in a novel situation. they involve not only understanding facts, but knowing how to use them. The following are examples of comprehension-based questions:

  • True/False: Using comparison or understanding of facts. For example:
    Forty two times three is one hundred twenty six.
  • Multiple Choice: Selection of the correct answer. For example:
    What is forty two times three?
    A. 39
    B. 45
    C. 126
    D. 14
  • Matching: Once again, like several multiple choices combined. For example:
    Match the equation with the answer:
    • A. 42 - 3
      B. 42 + 3
      C. 42 * 3
      D. 42 / 3      		 1. 39
      2. 45
      3. 126
      4. 14

    The key difference between reasoning and other types of questions is that you cannot memorize your way through a reasoning test. To answer these questions, you need to be able to perform multiplication, not just know what it means.
