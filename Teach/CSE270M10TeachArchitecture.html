CSE 270: W10 Plan: Usability Plan

Our lab this week will be to create a usability plan (sometimes called a usability test plan or even a usability study plan).

Usability Plan

Using the mobile application you and your partner(s) previously selected, create a usability plan. This will be accomplished by selecting a feature which you would like to study, list a set of instructions you would like the user to follow, and testing the plan with a friend to make sure it works as you expect.

Select a Feature

With your partner(s), select a task, feature, activity, or process which has a moderate degree of complexity. You are looking for an action that has about a dozen steps and more than one screen. For example, consider Instagram. An appropriate feature will be the enrollment process. This will consist of a new user adding a username and password to the system so the user can have access to Instagram's content.

Identify the Instructions

Once a feature has been selected, determine the set of instructions that the user will need to in order to accomplish the task. If the instructions are too vague, then it is unlikely that any two users will perform the same actions. If the instructions are too specific, then you will learn nothing about how a real user will interact with the system. The challenge is to give enough guidance that the user will accomplish the task, but not to over-specify.

The goal of this usability plan is to find defects in the feature of choice. In other words, will the average user be able to accomplish the task that the feature was designed to facilitate?

Share your instructions with the other members of your team. Debate which set of instructions will give you the best data on the feature which was previously selected.

Test the Plan

Using a friend, family member, or roommate, test your usability plan. Give your test subject the instructions as you have previously written them down. Observe them. Are they able to accomplish the task? Is there something you could have worded better? Do you need to be more or less specific? It may take a few rounds of testing to get the instructions right.

Submission

There will be one submission for the entire group. This submission will have a description of the feature and the instructions your team came up with.

Note that we will collect data and report on our findings later in the week.
