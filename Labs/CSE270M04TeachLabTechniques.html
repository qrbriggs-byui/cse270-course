Lab 04 - Determining Test Technique

Overview

Dr. Mobile will contain several types of features, sensors, emitters, and capabilities. Each one of these features will need to be tested. You have learned about more than a dozen different techniques that can be used from the code level up to the system level. You will be applying that knowledge to determine which technique is best to be used for different parts of Dr. Mobile.

Assignment

Read over existing posts on the Lab Discussion board.

Pick a 2 technqiues and 1 Dr. Mobile feature

You will need to pick 2 different testing techniques and one or more features of Dr. Mobile that have not been applied in the discussion board. If your techique was applied to a feature, you can't use that same technique for that feature.

Verification Post:

Create 2 different posts. Each post will be how Dr. Mobile feature will be tested with that technique. Identify the Lifecycle phase that the test will be used. Briefly describe the scope of the test and why this feature & technique would be important to verify.

Brainstorming Replies:

Yes, reply. Pick two of your classmates' posts. Identify either the Dr. Mobile feature or the testing technique. If you choose the feature, identify a different technique that could be used, provide reason to how this will help with the verification. If you choose the technique, identify a different feature that could also be tested using that technique.

Note: You can not use the same features or techninques you use in the Verification posts.

Submission

Post your Verification Posts and Verification Replies.
