Background

HAM radio is an amateur broadcasting radio network. It allows certified individuals to communicate via FCC regulated bandwidths. Some HAM operators have chosen to set up repeater towers to allow further communication over longer distances between locations. A portable handheld radio can connect to a repeater to increase its communication range. In some cases, if the repeaters have the same bandwidth, repeaters can be connected, extending the range of communication. The most common bandwidth ranges are:

  • 6 m (50 MHz) - Yes - Bandwidth (50-54 MHz)
  • 2 m (144 MHz) - Yes - Bandwidth (144-148 MHz)
  • 1.25 m (222 MHz) - Yes - Bandwidth (222-225 MHz) - Novice
  • 70 cm (440 MHz) - Yes - Bandwidth (420-450 MHz)
  • 33 cm (902 MHz) - Yes - Bandwidth (902-928 MHz)
  • 23 cm (1240 MHz) - Yes - Bandwidth (1240-1300 MHz) - Novice

For the sake of this lab, we will be focusing on the 2m (144-148 MHz) range.

How will the Repeaters communicate with each other?

The next step is to determine if the repeaters can connect.

"A repeater can’t receive and transmit at the same time on the same frequency, just like normal transceivers. This is called simplex. Since a repeater essentially needs to listen and repeat at once, they use two different frequencies that are slightly offset from each other. This is called duplex." (https://theprepared.com/survival-skills/guides/beginners-guide-amateur-ham-radio-preppers/#repeaters)

According to arrl.org, repeaters on the 2 Meter band use the following standard for duplex:

  • 146.01-146.37 Repeater inputs
  • 146.61-146.97 Repeater outputs
  • 147.00-147.39 Repeater outputs
  • 147.60-147.99 Repeater inputs

Using the table above, determine the input and outputs for each repeater. The first frequency is usually the output, and the frequency plus the offset is the input.
