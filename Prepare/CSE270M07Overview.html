CSE 270: W07 Overview

Generating a set of test cases that thoroughly and completely covers a feature is a key competency for any tester.

Objectives

By the end of the week, each student will be able to:

  • Appreciate the importance of good test cases in the testing process.
  • Generate high quality test cases.

Plan of the Week

This week, we will complete the following activities:

  1. Reading: Read about creating test plans. Focus on test cases, each of which contains all the information necessary to verify one aspect of the system under test.
  2. Select a Web Application: Identify the web application that you will test.
  3. Create Test Cases: Generate a dozen test cases to test your chosen web application.
  4. Demonstration: Create a video demonstrating the execution of your test cases.
