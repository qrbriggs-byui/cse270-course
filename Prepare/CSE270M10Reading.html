Select a Project

Our lab this week will be to conduct a simple usability test. In order to do this, it is necessary to find an application that we can work with.

With your partner(s), select a mobile application that everyone can put on their phone. This application should not be "mainstream;" the goal is that the average person will not have experience working with this application. For example Instagram would not be an appropriate choice. However, MountainProject (an application for rock climbers) might be.
