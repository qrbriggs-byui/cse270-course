CSE 270 Teacher Notes M08: Regression Testing

W08: Instructor Notes

Overview

This week is a follow on to Week 03 Methodology & 04 Techniques.
Focus of the week will be Regression Testing.

Objectives

By the end of the week, each student will be able to:

  • Differentiate Testing Level (Unit, Integration, Regression, System)
  • Develop a Regression Plan

Before the Week Begins (Prepare)

Preparation:

  • Have installed Visual Studio Community with Python and Selenium plug in Installed
  • Understand how TiddlyWiki works and access tags within the framework.
  • Post Annoucement.
  • Coordinate with TA.
    • Discuss struggling students and make plans to support and mentor.
    • Discuss W07 grading expectations.

Partnership/Group work:

  • The student will be working in groups to modify automated tests using the TiddlyWiki Dr. Mobile application.
  • Check Canvas “groups” to verify that all students are enrolled in a group.

During the Week (Teach)

Labs:

  • Dr. Mobile is to help Emergency Medical Personal with rescue and aid in times of disaster. There is a general workflow that any medical personal does to complete their task to stabilize a person.
  • Dr. Mobile is a similuated via Tiddlywiki. This simulation is going to be tested via Selenium automated test code.
  • Because there are minor python prerequisites for coding for this class, Students will be modifying the code.
